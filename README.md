## 개발시 이슈 사항
1. 기본 intellj  라이브러라가 인식 안될 경우.
   1. IntelliJ IDEA를 기본 SDK로 설정
      - IntelliJ IDEA에서 File -> Project Structure로 이동합니다.
      - 왼쪽 메뉴에서 SDKs를 클릭합니다.
      - 플러그인 개발에 사용하고자 하는 IntelliJ IDEA 버전이 목록에 없으면 + 버튼을 눌러 추가합니다.
      - Apply 또는 OK 버튼을 눌러 변경사항을 적용합니다.
